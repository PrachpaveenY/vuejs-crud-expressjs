## ขั้นตอนการ Run project
- Run command (Front-end) :
    ```
    $ cd vuejs-nodejs-crud-teach >> npm install >> npm run dev
    ```
- Run command (Back-end) :
    ```
    $ cd backend >> npm install >> npm run dev
    ```
